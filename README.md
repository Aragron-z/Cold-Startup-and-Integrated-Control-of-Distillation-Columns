# Cold-Startup-and-Integrated-Control-of-Distillation-Columns
本项目为过程控制课程设计，基于课程设计指导书完成精馏塔冷态开车与综合控制系统仿真。
通过虚拟仿真平台（PISP）模拟精馏塔从冷态到稳态运行的全过程，涵盖工艺流程认知、仪表识别、控制回路投运、安全联锁等内容。
 ## 🛠 项目功能
精馏塔工艺流程模拟：甲醇水混合物分离过程仿真；

P&ID图例学习：仪表位号识别与工艺流程图绘制；

冷态开车操作：按步骤完成精馏塔从进料到稳态运行的全程操作；

控制系统投运：包括流量、液位、温度等控制回路的自动投运与参数整定；

安全联锁与评分系统：理解安全联锁逻辑，完成实验评分。

 ## ⚙使用说明
 运行环境：流程工业智能仿真实训系统（PISP.exe）

 实验步骤概述：
 
1.启动PISP软件，加载“精馏塔冷态开车及综合控制实验”工程；

2.按照指导书步骤进行冷态开车操作；

3.依次投运进料流量、塔釜液位、回流流量、冷凝罐液位、灵敏板温度等控制系统；

4.PID参数整定。

评分系统：

控制指标：调节时间、稳态指标、误差性能指标；

累积量指标：甲醇产量；

安全指标：液位、压力等超限扣分；

总分由多个评分项综合计算得出。
